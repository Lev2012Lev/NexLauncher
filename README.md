# NexLauncher

A modern open-source Minecraft launcher written in C# and Avalonia.

## Goals

- Fast
- Cross-platform
- Easy for beginners
- Advanced customization
- QuickCSS support

## Roadmap

- [ ] Launcher UI
- [ ] Download Minecraft
- [ ] Launch Minecraft
- [ ] Fabric support
- [ ] Forge support
- [ ] Themes
- [ ] QuickCSS
